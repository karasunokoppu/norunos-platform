# Books (読書管理)

読んでいる本や読了した本を管理し、読書メモを残すことができます。

## 機能

1. **本棚**:
    * 登録した書籍がグリッド表示されます。
    * 表紙画像を設定して視覚的に管理できます。
2. **書籍の登録**:
    * 「Add Book」ボタンから、タイトル、著者、総ページ数などを登録します。
3. **読書メモ (Reading Memo)**:
    * 各書籍をクリックすると詳細画面が開き、「+ New Memo」からメモを追加できます。
    * **ページ数**を記録することで、その日の読書進捗がカレンダーに反映されます。
    * マークダウン形式で感想や要約を記録できます。
