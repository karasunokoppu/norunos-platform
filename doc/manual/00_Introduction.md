# はじめに

Norunos Platformへようこそ！
このアプリケーションは、日々のタスク管理、スケジュール調整、情報の整理をオールインワンで行うためのプラットフォームです。

## 主な機能

* **To Do List**: タスクの作成、管理、グループ分け。
* **Calendar**: タスクの期日確認、日別メモ、読書記録の確認。
* **Gantt Chart**: タスクの進捗状況を視覚的に把握。
* **Notes**: マークダウン形式での自由なノート作成とディレクトリ管理。
* **Books**: 書籍情報の管理と読書メモの作成。
* **Mind Map**: ノードベースの図解による思考整理。

## 画面構成

アプリケーションは左側のサイドバーと、右側のメインエリアで構成されています。
サイドバーから各機能への切り替えが可能です。
