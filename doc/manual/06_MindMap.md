# Mind Map (マインドマップ)

アイデア出しや思考整理のためのビジュアルツールです。

## 操作方法

1. **マップの作成**:
    * 「New Map」からタイトルを入力して新しいマップを作成します。
2. **ノードの操作**:
    * **追加**: 右上のパネルにある「+ Add Node」ボタンをクリックします。
    * **編集**: ノードを**ダブルクリック**すると、テキストを編集できます。
    * **削除**: ノードを選択して `Delete` または `Backspace` キーを押します。
    * **移動**: ノードをドラッグして好きな場所に配置できます。
3. **接続 (エッジ)**:
    * ノードの上下左右にあるハンドル（ドット）からドラッグして、他のノードへ線を繋げます。
    * 4方向すべてから自由に接続可能です。
4. **保存**:
    * 右上の「Save」ボタン、またはショートカットキー `Ctrl + S` で保存します。

## テーマ

アプリケーションのカラーテーマ設定（ダークモード等）に合わせて、ノードや背景の色が自動的に調整されます。
