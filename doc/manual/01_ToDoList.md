# To Do List (タスク管理)

日常のタスクを管理するための機能です。

## 基本操作

1. **タスクの追加**:
    * 画面下部の入力フィールドにタスク名を入力し、「Add」ボタンまたはEnterキーを押します。
    * 日付指定も可能です。
2. **タスクの完了**:
    * リスト上のチェックボックスをクリックすると、タスクが完了状態になります。
3. **タスクの編集**:
    * タスクをダブルクリックするか、コンテキストメニュー（右クリック）から編集可能です。
    * 詳細ダイアログで、メモや優先度、進捗率を設定できます。
4. **サブタスク**:
    * タスクにはサブタスクを追加でき、階層的に管理できます。

## グループ管理

* サイドバーの「Groups」からタスクグループを作成できます。
* タスクをグループにドラッグ＆ドロップすることで整理できます。
